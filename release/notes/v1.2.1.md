## What's changed

- **Linux arm64 builds.** `.deb`, `.rpm` and `.AppImage` are now published for arm64 alongside the existing x86_64 builds, built natively on GitHub's `ubuntu-24.04-arm` runner.
- **Snap now ships for arm64 too**, built on its matching native runner.
- Re-release of [v1.2.0](https://github.com/ZauJulio/ZeroWhats/releases/tag/v1.2.0) (auto-lock rewritten as a Rust-side idle watcher; fixed the notification app icon) — that tag was cut just before the arm64/CI changes landed, so this version carries the same app changes plus the completed release matrix.
